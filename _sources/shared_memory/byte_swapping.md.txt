# バイトスワップ

前述の通り、相手方のワード語長に従ってバイトスワップする必要があります。TwinCATでは、相手側のアーキテクチャのワードサイズ毎に、TwinCAT側（ホストバイトオーダ）に相互に変換するファンクションを用意しています。
